This is a project I started working on both to learn app development and to make something useful for botanical field work in Alberta. To get started press the menu in the top left corner and select 'Search' to be able to filter through the entire ACIMS species list. The search works by filtering any matches to the text entered, this queries Common, Scientific, and 7 letter code names.
